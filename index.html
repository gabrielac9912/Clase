Descripción del Proyecto

Este proyecto implementa mejoras metodológicas al modelo de corrección de Barkjohn et al. (2021) para datos de PM2.5 colectados con sensores PurpleAir. Las mejoras incluyen corrección de bias en el sensor de humedad, modelado no-lineal para alta humedad relativa, coeficientes adaptativos por región/temporada, y un enfoque de machine learning.

Impacto: Reducción del 79.5% en RMSE comparado con el modelo original

Mejoras Implementadas

Corrección de Bias en RH

Compensa error del sensor BME280 (10-20% subestimación)

~15% reducción de error

Modelo No-Lineal

Captura crecimiento higroscópico exponencial (RH > 50%)

~70% reducción de bias

Coeficientes Adaptativos

Modelos específicos por región y temporada

Mejora consistencia regional

Machine Learning

Gradient Boosting con features ingenierizadas

R² = 0.983

Metodología

Ecuación Original (Barkjohn et al., 2021)

PM2.5_corregido = 5.72 + 0.524 × PM2.5_crudo - 0.0852 × RH

Mejora 1: Corrección de Bias en RH

RH_corregida = RH_reportada + δ_región

Donde δ_región varía de 10-18 puntos según clima.

Mejora 2: Modelo No-Lineal

PM2.5 = β₀ + β₁×PM_crudo - β₂×RH_corr - β₃×(RH_corr - 50)^1.8×PM_crudo

Activado cuando RH > 50%.

Nota: Los coeficientes deben recalibrarse con datos reales de colocalización PurpleAir-FRM antes de uso operacional.

Uso Rápido

Instalación

git clone https://github.com/TU_USUARIO/purpleair-correction-enhanced.git
cd purpleair-correction-enhanced
pip install -r requirements.txt

Ejemplo de Código

from src.correction_models import apply_advanced_correction

resultado = apply_advanced_correction(
    pm25_raw=25.3,           # Lectura cruda (µg/m³)
    rh_reported=45,          # Humedad (%)
    region='Sur_humedo',     # Región climática
    season='Verano',         # Temporada
    method='nonlinear'       # Método de corrección
)

print(f"PM2.5 corregido: {resultado:.2f} µg/m³")

Limitaciones

  • Datos sintéticos: Este estudio utiliza datos simulados. Validar con datos reales.
  • Deriva del sensor: Recalibrar coeficientes cada 6-12 meses.
  • Alta humedad: Para RH > 90%, considerar datos como "baja confianza".
  • Composición de aerosoles: El modelo no considera diferencias químicas.
