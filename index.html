Discusión

4.1 Implicaciones de las Mejoras Implementadas

Corrección del Sensor de RH: Nuestros resultados confirman que el sesgo del sensor BME280 es una fuente significativa de error en la corrección de PM2.5. La subestimación de RH de 10-20 puntos porcentuales conduce a una corrección insuficiente del crecimiento higroscópico.

Recomendación práctica:

Los usuarios deberían aplicar una corrección de RH antes de usar cualquier ecuación de corrección de PM2.5, especialmente en regiones húmedas donde el error puede superar el 15%.

No-linealidad en Alta Humedad

La mejora más significativa se observa en rangos de RH > 50%, donde el crecimiento higroscópico de partículas se vuelve exponencial. El modelo lineal de Barkjohn no captura esta dinámica, llevando a errores sistemáticos crecientes que pueden alcanzar 5 µg/m³ en condiciones de RH > 90%.

Implicación para salud pública: Durante eventos de alta humedad (tormentas, temporadas de monzón), las lecturas no corregidas o corregidas con modelo lineal pueden llevar a alertas de calidad del aire falsamente elevadas.

Modelo Adaptativo vs. Machine Learning

Sorprendentemente, el modelo ML superó al modelo adaptativo paramétrico a pesar de este último estar específicamente calibrado por región/temporada. Esto sugiere que:

  • Las relaciones no-lineales capturadas por ML son más importantes que la estratificación regional
  • El modelo adaptativo puede sufrir de overfitting con datasets limitados por celda región-temporada
  • ML generaliza mejor a condiciones no vistas durante el entrenamiento

Limitaciones del Estudio

Datos Sintéticos

Este estudio utilizó datos sintéticos basados en parámetros empíricos de literatura. Si bien esto permite demostrar la metodología, los coeficientes finales deben ser recalibrados con datos reales de colocalización PurpleAir-FRM.

Trabajo futuro: Validar con ≥50 sitios colocalizados

Deriva del Sensor

No evaluamos la deriva temporal del sensor PurpleAir, la cual puede requerir recalibración periódica de los coeficientes cada 6-12 meses según Tryner et al. (2020).

Recomendación: Monitoreo continuo de calidad

Composición de Aerosoles

El modelo no considera diferencias en composición química de aerosoles (polvo mineral, sulfatos, carbono orgánico, humo de incendios), las cuales afectan la respuesta óptica del sensor.

Limitación: Modelo generalista

Representatividad Geográfica

Aunque mejoramos la distribución regional respecto al estudio original, el sureste húmedo permanece subrepresentado con solo 10% del dataset.

Necesidad: Más sitios en climas tropicales

Recomendaciones para Implementación

Para Agencias Regulatorias

  1. Adoptar corrección de RH: Implementar corrección del sesgo del sensor BME280 en todos los datos PurpleAir procesados
  2. Validación local: Realizar colocalización mínima de 3 meses antes de desplegar correcciones regionalizadas
  3. Monitoreo de calidad: Implementar alertas automáticas cuando RH > 90% (datos de baja confianza)

Para Investigadores

  • Transparencia: Publicar código y coeficientes en repositorios abiertos
  • Documentación: Registrar versión del modelo aplicado para reproducibilidad
  • Actualización: Recalibrar coeficientes anualmente o después de cambios de firmware del sensor

Para Usuarios del Público

  • Interpretación cautelosa: Datos en RH > 80% tienen mayor incertidumbre
  • Fuentes múltiples: Complementar con datos de AirNow para decisiones de salud
  • Actualizaciones: Verificar que la plataforma usada aplica la corrección más reciente

Conclusiones
